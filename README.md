# rtsx_usb

## Pre-requisites

### Debian-based distros
```
sudo apt-get install build-essential linux-headers-$(uname -r)
```

### Fedora
```
sudo dnf install kernel-devel

```

### Arch
```
sudo pacman -S linux-headers base-devel
```

## Download

```
git clone https://github.com/StarLabsLtd/rtsx_usb.git
cd rtsx_usb
```

## Install
```
make
```

## Uninstall
```
make uninstall
```
